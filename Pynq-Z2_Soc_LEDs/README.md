# Pynq-Z2 LED Blinking SDK
A simple LED blinking program, the Hello World of Digital, on the Pynq-Z2 board by the FPGA Processing System (PS). 

## Tools
<li>Vivado 2022.2</li>
<li>Vitis IDE</li>
<li>Pynq-Z2 Board</li>
